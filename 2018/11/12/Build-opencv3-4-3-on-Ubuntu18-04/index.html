Ubuntu18.04+Opencv3.4.3配置

发表于 2018-11-12 | 更新于 2018-11-13

1. 使用环境

  • Ubuntu 18.04.1 LTS

  • Opencv 3.4.3

  • Python 3.6.5 (Anaconda3 5.2)

2. 安装步骤

2.1 安装Anconda3

到官网下载对应安装包，如果要安装旧版，可以google或baidu清华的镜像源。

  • 建议配置国内的源，任何一个都可以

2.2 Opencv-python 3.4.3

2.3 编译OpenCV

如果你只想用python做opencv的开发，上面的工作完成后你就可以直接干活去了。这里我们想用C++开发，因此要编译Opencv。

  • windows版带有默认编译好的文件，VS开发时如果没用到高级功能是不需要再编译的。

  • linux的默认没有编译

2.3.1 下载源文件

  • 官网下载opencv 3.4.3(4.0截至目前还是beta版本) + opencv_contrib 3.4.3

  • 下载后提取到home目录下，会生成opencv3.4.3文件夹

  • opencv_contrib提取到opencv3.4.3文件夹根目录
